About this Project

This static site is designed for DevOps demos with Jenkins, Git/GitHub, and Docker.

  • Smoke tests: curl /health.
  • Caching: aggressive for assets, disabled for HTML.
  • Security: opinionated CSP + headers.

Use it for CI/CD tutorials, interviews, or internal demos.
